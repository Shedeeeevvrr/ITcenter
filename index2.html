Шаг №1

База в создании проекта

1.1 c/Напишите в файле !
и по первой подсказке нажмите Tab.
У вас появится форма, основа вашего проекта

1.2 c/разделим тело нашего проекта на три блока
<header>,
<main>
и <footer>

1.3 c/теперь сделаем три прострых действия для начала.
1/ изменим land c “eng” на “ru”.
2/ в строке <title> поменяем название со стандартного на “мой первый сайт”.
3/ в header напишем заголовок нашего сайта. <h1> Добро пожаловать на мой сайт</h1>

1.4 c/теперь вот так выглядит наш сайт
за его изменениями можно смотреть в живую
установив дополнение live server
либо вручную открыть файл с html

1.5 c/в main добавим меню навигации с помощью маркированного списка.

1.6 c/теперь всё должно выглядеть так. c/после навигационного меню добавьте текст
<h2>О нас</h2>
<p>Мы создаем удивительные веб-сайты.</p>

1.7 c/для окончания разметки сайта добавьте в footer описание<p> 2025 Мой первый сайт</p>

1.8 c/ на данный момент ваш сайт выглядит так. Но это только начало работы, далее начинается стилизация вашего сайта. Именно благодаря ней сайт приобритёт краски и смотрибельный внешний вид
